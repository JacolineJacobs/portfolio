Project Overview

SwapWisdom is a bold and dynamic web project designed to create a platform where users can connect, share knowledge, and collaborate seamlessly.

Focusing on three key pages—the user-friendly homepage, interactive user profiles, and an engaging contact page—this project allowed me to practice plugin integration and optimize user experiences with WordPress tools.

By combining strategic design choices with plugins like UsersWP, Better Messages, and WPForms, SwapWisdom delivers a smooth and intuitive experience tailored for knowledge-sharing communities.

Project Goals

SwapWisdom’s goal was to build an engaging and easy-to-navigate platform that connects users based on their skills and shared interests. I focused on:

  • Clean, Responsive Design:

    Crafting a homepage that looks great on any device and offers users a welcoming, intuitive entry point.

  • Streamlined Search and Filters:

    Developing a filtering system that allows users to find the skills they’re looking for without any hassle, improving their overall experience.

  • Customizable User Profiles:

    Leveraging UsersWP to create detailed user profiles where members can display their skills, interests, and experiences for easy connection opportunities.

Design Process

Inspiration and Observations

For SwapWisdom, I based my design decisions on my own experience navigating community-driven platforms and the goal to create a user-friendly space for skill sharing. I focused on the following design principles:

  • Simplicity and Clean Layouts:

    Drawing inspiration from minimalist designs, I aimed to keep the platform visually clear, reducing clutter and ensuring easy navigation. I wanted users to focus on the skills and connections they could make.

  • Responsive and Mobile-Friendly Design:

    Responsive and Mobile-Friendly Design: Given the mobile-first nature of today's web browsing, I ensured the platform worked seamlessly across devices, with particular attention to smaller screens.

  • Intuitive User Interface:

    I focused on creating an easy, frictionless experience, where users could quickly access the features they need, whether that's browsing skills, filtering options, or contacting others.

This approach allows you to describe your design decisions while still acknowledging that you focused on solving problems commonly found in other platforms or websites.

Colors & Typography

For Second Chapter, I carefully selected colors and fonts that would evoke a welcoming, modern feel while maintaining clarity and accessibility.

Colors:

  • Primary Color:

    A soft, earthy green (#7BAE7F) that gives a calm, vintage vibe—perfect for a secondhand book shop.

  • Light Color:

    A light, neutral tone (#ebeff0) to keep the background clean and easy on the eyes, offering good contrast with the primary and highlight colors.

  • Dark Color:

    A rich, deep charcoal (#01090a) for text and key elements, providing strong contrast and readability.

  • Highlight Color:

    A soft red (#FF6B6B) to draw attention to important buttons and calls to action, such as the "Add to Cart" button.

I also tested these colors for accessibility to ensure they would be clear and readable for all users.

Fonts:

  • Bitter:

    A serif font used for headings, providing a classic, literary feel that ties into the book theme.

  • Montserrat:

    A clean, modern sans-serif font used for body text, offering legibility and a contrast to the more decorative heading font.

The combination of these colors and fonts helps create a user-friendly, aesthetically pleasing design while maintaining accessibility standards.

Coding Skills Gained

In developing Second Chapter, I focused on building a solid foundation in both responsive design and JavaScript functionality to enhance the user experience. Here’s an overview of the key coding skills I applied and strengthened throughout the project:

    Responsive Design with HTML and CSS

  • Flexbox & Grid:

    Used to create a flexible layout that adjusts seamlessly across desktop and mobile views.

  • Media Queries:

    Applied CSS media queries to ensure that elements like product sections and navigation adapt smoothly to various screen sizes, enhancing the mobile-first experience.
    JavaScript Interactivity

  • Dynamic Cart Interaction:

    Implemented JavaScript to add and remove items in the cart, providing users with an interactive shopping experience

  • Responsive Navigation:

    Created a responsive menu using JavaScript to simplify navigation on smaller screens.

This project provided valuable hands-on experience with HTML, CSS, and JavaScript fundamentals, reinforcing my understanding of responsive web development and interactive features.

Conclusion

The Second Chapter project was an opportunity to apply my skills in responsive design and front-end development while focusing on creating an intuitive user experience for a secondhand book marketplace.

By addressing key usability issues—such as cluttered pages, inconsistent layouts, and lack of user feedback—I was able to create a more streamlined and engaging browsing experience.

The thoughtful use of color, typography, and mobile responsiveness ensured the site is both visually appealing and easy to navigate across all devices.

Overall, this project reinforced the importance of simplicity, accessibility, and user-centric design in building an effective online experience.
